はじめに

概要

このモジュールでは、OpenShift Virtualization を使い始めるために必要な基本ツールと基本的な設定について説明します。OpenShift Virtualization が初めての方でも、知識を再確認したい方でも、これらのガイドを活用して迅速に環境を構築できます。

学習内容

このセクションでは、以下の内容を学習します。

  • OpenShift Virtualization を利用するための前提条件と必要なツール

  • 仮想マシンの管理に使用する virtctl コマンドラインツールの基本的な使い方

  • 仮想マシンデプロイメントのためのデフォルトストレージの設定方法

  • cloud-init の基礎と VM 初期化のパターン

前提条件

このセクションのチュートリアルを進める前に、以下の準備ができていることを確認してください。

  • OpenShift Virtualization Operator がインストールされた OpenShift 4.18+ クラスター

  • クラスター管理者アクセス権限、または適切な RBAC 権限

  • インストール済みの CLI ツール: oc、kubectl、virtctl

  • Kubernetes/OpenShift の概念に関する基本的な理解
