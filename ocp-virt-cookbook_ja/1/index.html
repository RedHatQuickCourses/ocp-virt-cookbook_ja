OpenShift Virtualization クックブック

はじめに

OpenShift Virtualization でのワークロード開始およびデプロイに関する実践的なフィールドガイドです。

このクックブックでは、OpenShift を使い始めたばかりの方に向けて、すぐに役立つレシピ、マニフェストの例、およびトラブルシューティングのメモを提供します。本書は、理論的な学習に代わるものではなく、実際の導入を促進するための、公式ドキュメント および トレーニングコース を補完する 実践的なガイド として設計されています。

(最終更新日: {build-date})

前提条件

このクックブックを効果的に活用するには、以下が必要です。

  • クラスター: OpenShift Virtualization オペレーターがインストールされた OpenShift 4.18 以降。

  • ツール: oc、kubectl、および virtctl。（オプション: kustomize、helm、ansible）。

  • 知識: YAML および CLI ワークフローの習熟（必須ではありませんが、強く推奨します）。Kubernetes/OpenShift の概念に関する基本的な理解。

環境を準備または検証するために、特にこの技術が初めての場合は、はじめに の全モジュールを一通り確認してください。

クックブックのモジュール

左側のメニューからモジュールを 1 つ選択して開始してください。

  • はじめに - 必須ツールのセットアップと環境検証。

  • ストレージ設定 - 仮想マシンのストレージ設定。

  • ネットワーク構成 - ネットワークおよび IP 設定ガイド。

  • 仮想マシンの設定 - VM アクセス、DNS、ノード配置、およびテンプレート。

  • VM パフォーマンスチューニング - CPU ピニング、NUMA、Hugepages、およびリアルタイム VM。

  • VMライフサイクル管理 - ゴールデンイメージ、クローニング、スナップショット、およびブート設定。

  • YAML マニフェストライブラリ - すぐにデプロイ可能なマニフェスト例のライブラリ。

  • API 統合 - クローニング、エクスポート、マシン・トポロジーなどの API コンポーネントのアーキテクチャと機能。
