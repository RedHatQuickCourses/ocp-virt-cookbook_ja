= OpenShift Virtualization クックブック
:navtitle: はじめに

== はじめに

OpenShift Virtualization でのワークロード開始およびデプロイに関する実践的なフィールドガイドです。

このクックブックでは、OpenShift を使い始めたばかりの方に向けて、すぐに役立つレシピ、マニフェストの例、およびトラブルシューティングのメモを提供します。本書は、理論的な学習に代わるものではなく、実際の導入を促進するための、link:https://docs.redhat.com/en/documentation/openshift_container_platform/4.20/html-single/virtualization/index[公式ドキュメント,window=_blank] および link:https://www.redhat.com/en/services/training/do316-managing-virtual-machines-red-hat-openshift-virtualization[トレーニングコース,window=_blank] を補完する __実践的なガイド__ として設計されています。

(__最終更新日: {build-date}__)

== 前提条件

このクックブックを効果的に活用するには、以下が必要です。

* **クラスター**: OpenShift Virtualization オペレーターがインストールされた OpenShift 4.18 以降。
* **ツール**: `oc`、`kubectl`、および `virtctl`。（オプション: `kustomize`、`helm`、`ansible`）。
* **知識**: YAML および CLI ワークフローの習熟（必須ではありませんが、強く推奨します）。Kubernetes/OpenShift の概念に関する基本的な理解。

環境を準備または検証するために、特にこの技術が初めての場合は、**xref:getting-started:index.adoc[]** の全モジュールを一通り確認してください。
